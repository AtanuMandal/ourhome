# Onboarding a New Society

This document outlines the steps to onboard a new society into the system.

## Steps to Onboard a New Society

1. **Access the Admin Panel**
   - Log in to the admin panel using your credentials.

2. **Navigate to the Society Management Section**
   - Go to the "Society Management" section in the admin panel.

3. **Add New Society**
   - Click on the "Add New Society" button.
   - Fill in the required details for the new society, such as:
     - Society Name
     - Address
     - Contact Information
     - Number of Blocks/Buildings
     - Number of Apartments

4. **Configure Society Settings**
   - Set up the following configurations for the society:
     - Maintenance Fee Structure
     - Amenities and Facilities
     - Security and Access Control

5. **Add Apartments**
   - Add apartments to the society by following the steps outlined in the [Adding New Apartments](add_apartments.md) document.

6. **Assign Admins**
   - Assign society admins who will manage the society operations.
   - Provide them with the necessary access credentials.

7. **Verify and Publish**
   - Review all the details entered for the new society.
   - Once verified, publish the society to make it active in the system.

## Notes
- Ensure all mandatory fields are filled in before saving the society details.
- Double-check the contact information and address for accuracy.
- Inform the assigned admins about their roles and responsibilities.
