# Amenity Booking

## Overview
The amenity booking module will allow residents to book society amenities such as the clubhouse, gym, swimming pool, or community hall. This will ensure proper scheduling and avoid conflicts.

## Features

1. **Amenity Listing**
   - Display a list of available amenities with details (e.g., capacity, rules, availability).

2. **Booking System**
   - Residents can book amenities for specific time slots.
   - Option to cancel or reschedule bookings.

3. **Availability Calendar**
   - View a calendar showing available and booked slots for each amenity.

4. **Notifications**
   - Notify residents about booking confirmations, cancellations, or changes.

5. **Admin Features**
   - Approve or reject booking requests.
   - Set rules and restrictions for amenity usage.
   - Generate reports on amenity usage.
