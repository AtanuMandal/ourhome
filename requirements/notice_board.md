# Notice Board

## Overview
The notice board module will allow society management to post important notices and announcements for all residents to view. This will serve as a centralized communication platform for the society.

## Features

1. **Post Notices**
   - Society management can post notices with titles, descriptions, and attachments.
   - Notices can be categorized (e.g., maintenance, events, general announcements).

2. **View Notices**
   - Residents can view all posted notices.
   - Filter notices by category or date.

3. **Notifications**
   - Notify residents when a new notice is posted.
   - Option to mark notices as read/unread.

4. **Archive Notices**
   - Automatically archive old notices after a specified period.
   - Residents can access archived notices if needed.
